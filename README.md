# JavaScript Login Form

A simple login form project built with **HTML**, **CSS**, **JavaScript**, and **Bootstrap**. The main goal of this project is to practice DOM manipulation, form validation, event handling, and dynamic UI updates using vanilla JavaScript.

## Features

* User-friendly login form
* Real-time form validation
* Input field validation on focus loss
* Dynamic button enabling/disabling
* Custom visual feedback for invalid fields
* Welcome message after successful login
* Form hiding after successful authentication

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

## Project Structure

```
project-folder/

```

## How It Works

1. The user enters a name and password.
2. The submit button remains disabled until the required fields are filled.
3. Empty fields are highlighted when they lose focus.
4. When the form is submitted:

   * Invalid inputs trigger an alert message.
   * Valid inputs display a personalized welcome message.
   * The login form is hidden from the page.

## Learning Objectives

This project was created to practice:

* DOM selection and manipulation
* Event handling (`blur`, `submit`)
* Form validation
* Conditional statements
* Dynamic styling with JavaScript
* User interaction and feedback
